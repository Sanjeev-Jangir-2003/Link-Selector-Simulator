## Packages
zustand | State management for high-frequency simulation logic
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
- Simulation logic is client-side, using Zustand for state management to handle the simulation loop and metrics.
- Recharts is used for data visualization with a custom dark/neon theme.
- Framer Motion is used for the network visualization animations.
